Plantie

This is a web application that I developed to learn more about Django before inheriting a project from a previous team. I learned how to use Django Rest Framework, SQL & MySQL databases, Google Authentication , and application deployment. Off all projects that I could have done, I decided to build a social app that allows individuals to post pictures of their plants. Music Compser

This is a desktop application that I developed with 3 other teammates. We were given specifications for how the final application was supposed to function, and we implemented those specifications. I learned how to use JavaFX, XML and improved my Java skills. To ensure smooth and professional version control, I utilized GitHub. Some of the features of the application include the ability to play, save, and load music. You can also make your own musical compositions using the provided instruments (piano, guitar, trumpet, et al). Copy, Cut, Paste, Select, Select All Undo & Redo buttons are also available. Flask - Weather API

This is a Flask web application that retrieves and displays weather information for a specified location using Visual Crossing Weather API. I learned how to use Flask-limiter to control the frequency of API calls from users, and Flask routing to manage https POST and GET requests. Error handling for API calls and invalid location are addressed.

Cache Proxy CLI (Python)

Cache Proxy server uses Python sockets to cache and serve HTTP requests efficiently. By storing responses, it reduces load times and improves performance, especially for repeated requests. TimeTour

This counts as my introduction to web development. Timetour allows you to convert time from one timezone to another, and knowing how far a specific date is. I utilized JavaScript, HTML and CSS. In addition, I used a Rest API from timezonebd for converting the timezones. I was inspired to develop this project because my family is in South Africa, and I am in the United States, and I wanted to know the time difference between us.
